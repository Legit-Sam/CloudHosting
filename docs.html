Introduction

Applications and workloads have different infrastructure needs
and may be spread across multiple bare-metal servers and public or private clouds,
making managemen complex, time-consuming, and costly. Cloud simplifies infrastructure management by
allowing you to deploy clusters anywhere on demand with one consistent experience,
reducing operational overhead and making your developers more productive. The self-service portal enables developers to create, observe, and manage
their own Kubernetes clusters within established guardrails so they can ship code faster.

    Install CLI

    Requirements

    • Windows 10, Mac OSX, Linux
    • Node.js v12 or higher

    Install

    Mac (Homebrew)

    $ brew install SamWeb-cli

    NPM

    $ npm install SamWeb-cli

    Yarn

    $ yarn install SamWeb-cli
